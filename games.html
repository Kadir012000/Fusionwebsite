Last Standing

5v5 Tactical First-Person Shooter

About the Game

Last Standing is a 5v5 tactical FPS built around a plant-and-defuse objective. Matches are round-based and require coordination, map knowledge, and mechanical skill.

The game is designed with competitive play in mind, focusing on clear mechanics, predictable gun behavior, and readable environments.

Core Features

  • 5v5 round-based matches
  • Plant and defuse objective mode
  • Manual recoil and movement-based accuracy
  • Ranked and unranked matchmaking
  • Private custom matches

Game Modes

  • Competitive — ranked matchmaking
  • Casual — unranked matches
  • Custom — private lobbies

Development status: Prototype / early alpha
